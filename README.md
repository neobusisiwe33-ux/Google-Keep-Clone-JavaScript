
This project is a simplified Google Keep Clone built with HTML, CSS, and JavaScript. It allows users to create, display, and archive notes through a clean and responsive interface while demonstrating core front-end web development skills.

Features
Create new notes
Display saved notes
Archive notes
User-friendly design
Interactive 

Code Used
HTML5
CSS3
JavaScript
